# Jupiter Broadcasting's Network
A catch-all for bugs and feature requests across the entire Jupiter Broadcasting network.

---

If you notice something that's 🐞 amiss, or you have a great 💡 idea for a new feature - let us know!

This includes third-party sites associated with or used by Jupiter Broadcasting, from podcast directories to social media sites.


For issues with certain services please file an issues in their respective repo:
* [Jupiter Broadcasting's website](https://github.com/JupiterBroadcasting/jupiterbroadcasting.com)
* [JB's Matrix server](https://github.com/JupiterBroadcasting/matrix)
* [JB's Peertube instance - Jupiter.tube](https://github.com/JupiterBroadcasting/jupiter.tube)
